  • deployd stuck on starting deployd v0.6.xx.. OSX [fix]
    2014 Mar

    The Problemo

    Yesterday I was checking deployd and wanted to try it out. Doing as the docs said, when I to started deployd It stuck on starting deployd v0.6.10...

    ##The Fixo

    After going through the issues on github i finally found this old-ish similar issue which was still unresolved. To solve it start monogd as a seperate process and run deployd as dpd -P 27017 -H localhost -n testing-deployd -d
