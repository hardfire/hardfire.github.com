GIT The Stupid Content Tracker

  • Distrubuted Version Control System

  • Initially designed and developed by Linus Torvalds for Linux kernel development

    I'm an egotistical bastard, and I name all my projects after myself. First 'Linux', now 'git'

    Linus Trovalds

  • Simple & Fast

Getting Git Pretty Easy

Ubuntu
# apt-get install git

Mac
$ brew install git
http://www.git-scm.com/download/mac

Win
http://www.git-scm.com/download/win (exe)

Configuring git config

git config --global user.name "Avinash Kundaliya"

git config --global user.email "avinash@avinash.com.np"

More on https://help.github.com/articles/set-up-git

Working with remote

  • git clone

  • git push <remote> <branch>

  • git fetch <remote>

  • git pull <remote>

  • git remote add origin git@github.com:hardfire/foo.git

Things that people say

  • Commit early, commit often

  • Always inspect your changes before committing

  • Write every commit message like the next person who reads it is an axe-wielding maniac who knows where you live

  • commit your own changes – you can’t delegate it
