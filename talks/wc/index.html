wp_register_script

Yo WP! Remember this script

($handle,$src,$deps,$ver,$in_footer);
  • $handle - Name of the script
  • $src - URL to the script
  • $deps - Array of dependencies
  • $ver - Version of the code
  • $in_footer - Place the script tag in footer

wp_register_script - example

 wp_register_script (
  'mytheme-custom', /* handle WP will know JS by */
  get_template_directory_uri() . '/js/custom.js', /* source */
  array('jquery'), /* requires jQuery */
  1.0, /* version 1.0 */
  true /* can load in footer */
);

Note: Don't hardcode the directory paths, use functions like plugins_url or get_template_directory_uri

available libraries : jquery , scriptaculous-root , jquery-ui-code , jquery-form ... and lot more

wp_deregister_script

Yo WP! forget about this script

 
	wp_deregister_script('jquery');

wp_enqueue_script

Yo WP! Please put the script in my page

 
wp_enqueue_script( $handle ,$src ,$deps ,$ver ,$in_footer );

A little complex example

 
function my_scripts_method() {
    wp_deregister_script( 'jquery' );
    wp_register_script( 'jquery', 'http://ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js');
    wp_enqueue_script( 'jquery' );
}    
 
add_action('wp_enqueue_scripts', 'my_scripts_method');

Use jQuery from google CDN instead of WordPress local

wp_dequeue_script

Hey WP! Someone put this script in my page, remove it please

 
	wp_dequeue_script($handle);
